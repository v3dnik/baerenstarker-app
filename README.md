# 🐻 Bärenstarker Transport GmbH — Rechnungs-App

## 📱 Namestitev na telefon (PWA) — korak za korakom

### Korak 1: GitHub račun
1. Pojdite na **github.com** in kliknite "Sign up" (brezplačno)
2. Ustvarite račun z vašim emailom

### Korak 2: Naložite kodo na GitHub
1. Po prijavi kliknite **"+ New repository"**
2. Ime repozitorija: `baerenstarker-app`
3. Izberite **Public** in kliknite "Create repository"
4. Kliknite **"uploading an existing file"**
5. Povlecite VSE datoteke iz te mape in kliknite "Commit changes"

### Korak 3: Vercel deployment (brezplačno)
1. Pojdite na **vercel.com** in kliknite "Sign up"
2. Izberite **"Continue with GitHub"**
3. Kliknite **"Add New Project"**
4. Izberite vaš repozitorij `baerenstarker-app`
5. Vse nastavitve pustite privzete — kliknite **"Deploy"**
6. Čez ~1 minuto dobite link npr: `baerenstarker-app.vercel.app`

### Korak 4: Dodajte na telefon

**iPhone (Safari):**
1. Odprite link v **Safari** (ne Chrome!)
2. Tapnite ikono **Skupna raba** (kvadrat s puščico)
3. Izberite **"Dodaj na začetni zaslon"**
4. Aplikacija se pojavi kot ikona na vašem zaslonu ✅

**Android (Chrome):**
1. Odprite link v **Chrome**
2. Tapnite meni (3 pike)
3. Izberite **"Dodaj na začetni zaslon"**
4. Aplikacija deluje kot prava app ✅

## 🔄 Posodobitve
Ko spremenite kodo in jo naložite na GitHub, se Vercel **samodejno posodobi** v ~1 minuti.

## 💡 Funkcije aplikacije
- ✅ Rachnung / Auftragsbestätigung / Offerte
- ✅ Stranke (Herr/Frau/Firma)
- ✅ Pozicije: Transport, Umzug, Räumung, Entsorgung, Montage, Reinigung, Sonstiges
- ✅ Ure + urna postavka → samodejni izračun
- ✅ MwSt. 8.1% (CH)
- ✅ Zahlungsstatus: Offen / Bezahlt
- ✅ Zahlungsart: Barzahlung / Kartenzahlung / Überweisung
- ✅ Quittung z "Betrag dankend erhalten" ko je plačano
- ✅ Übersicht z filtri in statističnimi povzetki
- ✅ PDF / Tisk
- ✅ Shranjuje podatke lokalno na napravi
